# Restaurant POS - Waiter Mobile App 📱

هذا هو تطبيق الموبايل الخاص بالنادل، تم بناؤه باستخدام **React Native (Expo)**.

## المميزات:
1. **اتصال مباشر بسحاب Supabase:** يتم جلب المنيو وإرسال الطلبات لنفس قاعدة بيانات تطبيق الويب.
2. **واجهة مستخدم سريعة:** مصممة خصيصاً للعمل باللمس على الهواتف.
3. **مزامنة فورية:** تظهر الطلبات التي يرسلها النادل فوراً في شاشة المطبخ (Kitchen Display).

## طريقة التشغيل:

1. **تثبيت المتطلبات:**
   ```bash
   cd mobile-waiter
   npm install
   ```

2. **بدء التطبيق:**
   ```bash
   npx expo start
   ```

3. **للتجربة على هاتفك:**
   - قم بفتح تطبيق **Expo Go** على هاتفك (Android/iOS).
   - امسح كود الـ QR الظاهر في التيرمينال.

## ملاحظة تقنية:
- يستخدم التطبيق ملف `src/lib/supabase.js` للاتصال بقاعدة البيانات.
- تم نقل منطق الـ Logic من مشروع الويب لضمان توافق الحسابات والأسعار.
